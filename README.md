Como usar Github
Para crear una cuenta GitHub vamos a seguir los siguientes pasos:

Entrá a GitHub.

Finalmente, solo habrá que llenar un pequeño formulario con los siguientes datos:
Nombre de usuario.
Dirección email.
Contraseña.
Confirmar contraseña.
